## 模块详解

### 1. 预处理模块 (src/preprocessor/)

| 组件 | 文件 | 功能 |
|------|------|------|
| FastaParser | fasta_parser.py | 解析FASTA格式序列文件 |
| PdbValidator | pdb_validator.py | 验证PDB结构文件完整性 |
| DataMapper | data_mapper.py | 建立序列与结构的映射关系 |

### 2. 预测器模块 (src/predictors/)

| 组件 | 文件 | 功能 |
|------|------|------|
| BasePredictor | base_predictor.py | 预测器基类，定义接口 |
| BepipredWrapper | bepipred_wrapper.py | 包装BepiPred-3.0 |
| DiscotopeWrapper | discotope_wrapper.py | 包装DiscoTope-3.0 |
| GraphbepiWrapper | graphbepi_wrapper.py | 包装GraphBepi |
| EpigraphWrapper | epigraph_wrapper.py | 包装EpiGraph |
| VaxignmlWrapper | vaxignml_wrapper.py | 包装Vaxign-ML (Docker) |

### 3. 聚合模块 (src/aggregator/)

| 组件 | 文件 | 功能 |
|------|------|------|
| ConsensusScorer | consensus_scorer.py | 计算共识评分 |
| ImmunogenicityEvaluator | immunogenicity.py | 评估免疫原性 |
| CandidateRanker | candidate_ranker.py | 候选排序 |

### 4. 报告模块 (src/reporter/)

| 组件 | 文件 | 功能 |
|------|------|------|
| CsvExporter | csv_exporter.py | 导出CSV结果 |
| HtmlReporter | html_report.py | 生成HTML报告 |

## 数据流

```
FASTA + PDB → 预处理 → 5个预测器并行预测 → 结果标准化 
           → 共识评分 + 免疫原性评估 → 综合排序 → 输出报告
```

## 环境隔离策略

由于各模型依赖版本不同，采用多Conda环境隔离：

```
master_env (Python 3.9)      ← 主控环境
    │
    ├── bepipred_env (Python 3.8.8, PyTorch 1.12)
    ├── discotope_env (Python 3.9, PyTorch 1.11)
    ├── graphbepi_env (Python 3.9.12, PyTorch 1.12.1)
    ├── epigraph_env (Python 3.8.12, PyTorch 1.10.1)
    └── vaxignml (Docker容器)
```

通过 `subprocess` + `conda run` 实现环境切换。
